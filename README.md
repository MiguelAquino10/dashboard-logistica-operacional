# Dashboard Operacional de Logística

## Contexto
Projeto desenvolvido para acompanhamento diário de veículos em operação logística,
com foco na visualização rápida de status e segmentação por tipo de doca.

## Problema
A operação não possuía um painel consolidado para identificar:
- total de carros em curso
- carros por tipo de doca (principal x telescópica)
- status de hubs

## Solução
Criação de um relatório em Excel com:
- base de dados estruturada
- regras de negócio por faixa de docas
- KPIs automáticos usando fórmulas

## Tecnologias e conceitos
- Excel avançado
- CONT.SES, SE, modelagem de dados
- KPIs operacionais
- Lógica de classificação

## Resultado
Relatório utilizado diariamente para tomada de decisão operacional.
